# FifaRegex-App

Website which shows reports of FIFA soccer matches. This project is built with vanilla JS and regular expressions are used to filter the information of a text file with reports of more than 3900 lines.

## Applied Concepts

- Regular Expressions

## Make with

- Javascritp
- Html
- Css
